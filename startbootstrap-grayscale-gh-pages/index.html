About Outrun Jules Verne

Outrun Jules Verne is a data science project to personalize travels. You get to pick features below based on your whim, and I get to recommend you a route around the globe!

Specifically, this model uses a fun combination of Natural Language Processing for feature extraction and Monte Carlo Metropolis for some randomness. For some more information, go to my github page!

Technology used: Python, Spark, Hadoop, AWS, and Bootstrap - Greyscale.
